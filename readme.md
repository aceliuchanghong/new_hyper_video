### 使用

```bash
# 技能安装
npx hyperframes skills
# 为 claude code 执行
npx skills add heygen-com/hyperframes

cd 00_测试系列
# 在当前现有文件夹内进行初始化
npx hyperframes init .
# 重新唤醒预览服务
npx hyperframes preview
# 渲染成 MP4
npx hyperframes render --output output.mp4
```
